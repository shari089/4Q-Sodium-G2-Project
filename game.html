Objectives

Find all matching pairs of cards within the given time limit.

Difficulty Levels

Easy Version: Picture Matching

Hard Version: Scientist-invention matching

How to Play

Flip two cards and match identical or scientist‑invention pairs within the time‑limit.

Rules & Constraints

Card Selection: Only two cards at a time can be selected.

Matching: Easy mode pairs must be identical. Hard mode pairs must be inventor to invention.

Time Limit: Easy mode starts at 60 seconds. Hard mode starts at 90 seconds.

Time Penalty: Score cannot go below 0, so a penalty of -2 seconds in time is applied if score tries to go below 0.

Scoring System

Correct: +10 points

Incorrect: -2 points

Player Count

Single‑Player Mode: The game is designed for only one player.
