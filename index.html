Mechanics and Win Conditions

Card Flip Mechanic: Players flip cards to reveal pictures or scientists and inventions.

Matching Mechanic: Players match identical pictures or scientists with their corresponding inventions.

Time Limit Mechanic: The game ends when the time limit is reached.

Win Condition

Easy Version: Find all matching pairs before time runs out.

Hard Version: Achieve a high score without reaching the Game Over Threshold.

Game Over Threshold

Easy Version: Not applicable.

Hard Version: Game ends if the player's score reaches -20 or they correctly match all pairs.

Rules and Constraints

Card Selection: Players can only select two cards at a time.

Matching Rules: Pictures or scientists and inventions must match exactly.

Time Limit: The game has a set time limit.

Scoring System

Correct Match: +10 points

Incorrect Match: -2 points

Combo Bonus: +15 points for every 3 correct matches in a row
